# GB/T 9704《党政机关公文格式》排版规范摘要

> 来源：GB/T 9704-2012《党政机关公文格式》国家标准
> 整理日期：2026-04-03
> 用途：本插件字体配置的核心依据

## 一、公文要素字体字号规范

### 标题与正文

| 要素 | 字体 | 字号 | 磅值 | 加粗 | 本插件对应 |
|------|------|------|------|------|-----------|
| 公文标题 | 方正小标宋简体 | 二号 | 22pt | 否 | `# H1` |
| 一级标题 | 黑体 | 三号 | 16pt | 否 | `## H2` |
| 二级标题 | 楷体_GB2312 | 三号 | 16pt | 否 | `### H3` |
| 三级标题 | 仿宋_GB2312 | 三号 | 16pt | **是** | `#### H4` |
| 四级标题 | 仿宋_GB2312 | 三号 | 16pt | 否 | `##### H5` |
| 正文 | 仿宋_GB2312 | 三号 | 16pt | 否 | 正文段落 |

### 其他要素

| 要素 | 字体 | 字号 | 磅值 |
|------|------|------|------|
| 份号 | 仿宋_GB2312 | 三号 | 16pt |
| 密级和保密期限 | 黑体 | 三号 | 16pt |
| 紧急程度 | 黑体 | 三号 | 16pt |
| 发文机关标志 | 方正小标宋简体 | — | 由发文机关确定 |
| 发文字号 | 仿宋_GB2312 | 三号 | 16pt |
| 签发人姓名 | 楷体_GB2312 | 三号 | 16pt |
| 附件说明 | 仿宋_GB2312 | 三号 | 16pt |
| 版记（印发机关） | 仿宋_GB2312 | 四号 | 14pt |

### 关键规则

1. **除公文标题使用二号字（22pt）外，正文和各级标题均使用三号字（16pt）**
2. 各级标题的区分 **仅靠字体和加粗**，不靠字号大小
3. 标题编号格式：一、→ (一) → 1. → (1)
4. 正文行距：一般为 28 磅固定行距（28/16 ≈ 1.75 倍行高）

### VS Code 中的对应配置

- `editor.fontSize: 22`——对应三号字
- `editor.lineHeight: 1.8`——对应 28 磅行距（28/16 ≈ 1.75）
- `editor.wordWrapColumn: 56`——28 个中文全角字符 = 56 个等宽列
- `editor.wordWrap: "bounded"`——取视口宽度和 56 列的较小值

## 三、本插件的实现映射

### Markdown 层级 → 公文层级

```
# H1  →  公文标题（二号 22pt，小标宋/宋体粗体）
## H2  →  一级标题（三号 16pt，黑体）
### H3  →  二级标题（三号 16pt，楷体）
#### H4  →  三级标题（三号 16pt，仿宋加粗）
##### H5  →  四级标题（三号 16pt，仿宋）
正文段落  →  正文（三号 16pt，宋体/仿宋）
```

### CSS 百分比计算

以 `editor.fontSize: 22`（对应公文三号字 22px）为基准：

| 公文要素 | 实际字号 | 百分比 | 计算方式 |
|---------|---------|--------|---------|
| 公文标题 | 二号 22pt | **137%** | 22/16 × 100% |
| 一级标题 | 三号 16pt | **100%** | 16/16 × 100% |
| 二级标题 | 三号 16pt | **100%** | 16/16 × 100% |
| 三级标题 | 三号 16pt | **100%** | 16/16 × 100% |
| 四级标题 | 三号 16pt | **100%** | 16/16 × 100% |
| 正文 | 三号 16pt | **100%** | 基准 |

> **注**：百分比是相对于 VS Code 编辑器基础字号的比例。当 `editor.fontSize: 22` 时，100% = 22px ≈ 三号字，137% = 30px ≈ 二号字。
